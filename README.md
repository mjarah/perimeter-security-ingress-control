# Perimeter Security Orchestration & Ingress Control

## Executive Summary
Implemented and validated host-based firewall rules to restrict
unauthorized ingress traffic while maintaining authorized access.

## Environment
- OS: Ubuntu Linux
- Tools: UFW, Nmap

## Objective
Apply least-privilege ingress controls at the system perimeter.

## Implementation
- Configured firewall default deny rules
- Allowed required service ports
- Persisted configuration across reboots

## Validation
- Confirmed authorized access
- Verified blocked unauthorized traffic
