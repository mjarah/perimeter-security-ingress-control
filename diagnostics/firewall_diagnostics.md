## Issue
Firewall rules blocked SSH access unintentionally.

## Diagnosis
Default deny rule applied before SSH allow rule.

## Resolution
Reordered firewall rules to allow SSH prior to deny-all.

## Verification
Re-tested SSH access and confirmed success.
